# Day 1 - Linux Basics

## Commands I learned

### echo
Used to print text or values in the terminal.
Example:
echo Hello

---

### whoami
Shows the current logged-in user.

---

### id
Displays user ID, group ID, and groups you belong to.

---

### root
The super administrator of the system with full control.

---

### htop
A real-time system monitor that shows CPU, memory usage, and running processes.

---

### apt
Package manager for Debian-based systems like Ubuntu.

sudo apt update  
Updates the package list.

---

### sudo
Stands for "Super User Do" and runs commands with administrator privileges.

---

### pwd
Prints the current directory path.

---

### ~ (tilde)
Represents the home directory.

echo ~  
Shows home directory path.

---

### ls
Lists files and folders.

ls → current directory  
ls ~ → home directory

---

## What I understood well
Linux commands are simple but powerful. Everything is organized using users, groups, and permissions.

---

## What I found interesting
How root has full control and how Linux uses a terminal for almost everything.

---

## What I found confusing
you have to memorise the command and its casesensitive.
